## v0.1 Scope

TimeWeaver v0.1 focuses on turning a vague goal and limited time into a realistic, executable plan.

### Included

- **Single task input**
  - Task name
  - Total available time (hours)
  - Planning period (days)
  - Daily time limit

- **Automatic task decomposition**
  - Split total work into reasonable time blocks
  - Default block duration
  - Built-in buffer to avoid over-planning
  - Avoid overly fragmented blocks

- **Time allocation**
  - Assign time blocks sequentially across days
  - Respect daily time limits
  - Carry over overflow automatically

- **Calendar view (read-only)**
  - Simple weekly or multi-day view
  - Display generated time blocks per day
  - No manual editing or drag-and-drop

- **Local persistence**
  - Save and load plans as human-readable JSON
  - No cloud, no accounts

### Explicitly Out of Scope

- Multiple tasks or task dependencies
- Notifications or reminders
- AI-based task understanding
- Precise scheduling to specific clock times
- Cloud sync or user accounts
- Polished or animated UI

v0.1 prioritizes planning logic and clarity over features and appearance.
